# ぬるぬる 💚

LINE風のNostrクライアント

## ✨ 機能

- **🏠 ホーム** - プロフィールの表示・編集、自分の投稿一覧
- **💬 トーク** - NIP-17 暗号化DM（NIP-44/59対応）
- **📱 タイムライン** - リアルタイム投稿表示、いいね・リポスト・Zap
- **⚡ ウォレット** - NWC (Nostr Wallet Connect) 設定

## 🛠️ セットアップ

```bash
npm install
npm run dev
```

## 🚀 Vercelデプロイ

1. GitHubにプッシュ
2. Vercelでインポート
3. Deploy!

## 📋 技術スタック

- Next.js 14
- React 18
- Tailwind CSS
- nostr-tools

## 📝 NIPs対応

- NIP-01: Basic protocol
- NIP-07: ブラウザ拡張機能ログイン
- NIP-17: Private Direct Messages
- NIP-44: Encrypted Payloads
- NIP-59: Gift Wrap
- NIP-47: Nostr Wallet Connect
